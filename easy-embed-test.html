Google Analytics風 簡単導入テスト

🚀 タグ1つで簡単導入

💻 簡単導入

Google AnalyticsやGTMのように、1つのタグを貼り付けるだけで導入完了。複雑な設定は不要です。

⚡ 高速動作

軽量で高速な動作を実現。ページの読み込み速度に影響を与えません。

📱 レスポンシブ

PC、タブレット、スマートフォンすべてに対応。どのデバイスでも快適に利用できます。

🎨 カスタマイズ

色やポジションなどを簡単にカスタマイズ可能。サイトデザインに合わせて調整できます。

📋 導入手順

  1. バックエンドサーバーを起動
    node server.js でサーバーを起動してください。
  2. タグを貼り付け
    下記のコードをHTMLの </body> タグの直前に貼り付けてください。
  3. 設定の調整
    必要に応じて apiUrl や色などの設定を調整してください。
  4. 動作確認
    右下にチャットボタンが表示されることを確認してください。

🔧 基本の導入コード

最もシンプルな導入方法（ローカル環境用）：

<script src="toneya-chatbot.js"></script>
<script>
  ToneyaChatbot.init({
    apiUrl: 'http://localhost:3000'
  });
</script>

🌐 本番環境用コード

本番環境での導入コード例：

<script src="https://your-cdn.com/toneya-chatbot.js"></script>
<script>
  ToneyaChatbot.init({
    apiUrl: 'https://your-api.com'
  });
</script>

⚙️ カスタマイズ例

色やポジションをカスタマイズした例：

<script src="toneya-chatbot.js"></script>
<script>
  ToneyaChatbot.init({
    apiUrl: 'http://localhost:3000',
    position: 'bottom-left',
    primaryColor: '#e53e3e',
    companyName: 'カスタム葬儀社',
    phoneNumber: '0120-123-456'
  });
</script>

🔗 URL パラメーター自動初期化

Google Analytics風の自動初期化：

<script src="toneya-chatbot.js?apiUrl=http://localhost:3000"></script>
